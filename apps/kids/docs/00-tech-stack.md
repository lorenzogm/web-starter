# Tech Stack - Aplicación de Matemáticas para Niños

Tecnologías y herramientas utilizadas en la aplicación.

## Frontend Framework

- **React 19** - Biblioteca de interfaz de usuario
- **TypeScript 5.7** - Tipado estático

## Routing

- **TanStack Router v1.87** - Enrutamiento con tipado y file-based routing

## UI & Styling

- **Tailwind CSS v4** - Framework de utilidades CSS
- **React Aria Components** - Componentes accesibles sin estilos
- **shadcn/ui** - Componentes reutilizables (estilo New York, base neutral)
- **lucide-react** - Biblioteca de iconos
- **class-variance-authority** - Gestión de variantes de componentes
- **clsx** - Utilidad para clases condicionales
- **tailwind-merge** - Fusión inteligente de clases Tailwind

## Build Tools

- **Vite v6** - Herramienta de compilación y desarrollo
- **PostCSS** - Procesador CSS
- **autoprefixer** - Prefijos CSS automáticos

## Testing

- **Vitest** - Framework de testing unitario

## Code Quality

- **ESLint** - Linter de código
- **Prettier** - Formateador de código
- **TypeScript** - Verificación de tipos

## Package Manager

- **pnpm** - Gestor de paquetes

## Estructura de Archivos

- File-based routing con TanStack Router
- Componentes UI en `src/components/ui/`
- Rutas en `src/routes/`
- Tipos en `src/types/`
- Datos en `src/data/`
